# Команды

### Менеджер пакетов pacman

{% code title="Обновить все пакеты" overflow="wrap" %}
```bash
sudo pacman -Syu
```
{% endcode %}

{% code title="Установить пакет" overflow="wrap" %}
```bash
sudo pacman -S ...
```
{% endcode %}

{% code title="Найти пакет" overflow="wrap" %}
```bash
sudo pacman -Ss ...
```
{% endcode %}

{% code title="Удалить пакет" overflow="wrap" %}
```bash
sudo pacman -Rsc ...
```
{% endcode %}
