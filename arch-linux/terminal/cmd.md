# Базовые команды терминала

### Менеджер пакетов pacman

Обновить все пакеты

{% code overflow="wrap" %}
```bash
sudo pacman -Syu
```
{% endcode %}

Установить пакет

{% code overflow="wrap" %}
```bash
sudo pacman -S ...
```
{% endcode %}

Найти пакет

{% code overflow="wrap" %}
```bash
sudo pacman -Ss ...
```
{% endcode %}

Удалить пакет

{% code overflow="wrap" %}
```bash
sudo pacman -Rsc ...
```
{% endcode %}
